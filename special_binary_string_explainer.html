2 The Algorithm: Recursive Decomposition ▼
1Define base case(s)
2Make recursive calls on smaller subproblem
3Combine results
4Return to caller
3 Why It Works: The Invariant ▼
🔐 The Invariant

Base case terminates recursion. Recursive case reduces problem size. Return value propagates up.

4 The Code (Annotated) ▼
def mergeTwoLists(l1, l2):
if not l1: return l2
if not l2: return l1
if l1.val <= l2.val:
l1.next = mergeTwoLists(l1.next, l2)
return l1
else:
l2.next = mergeTwoLists(l1, l2.next)
return l2
