LeetCode #1525 – Count split positions where both halves have equal distinct char counts

Medium Prefix-Suffix Distinct Counts O(n) Time
1Understanding the Problem▼

Split string s at every position i into left = s[0..i] and right = s[i+1..n-1]. Count positions where distinct(left) == distinct(right).

Example: s="aacaba" → valid splits at indices where both halves have same number of distinct chars.

💡 Key Insight

Precompute prefix and suffix distinct counts in two passes. Then a single scan checks each split point in O(1).

2The Algorithm: Prefix-Suffix Distinct Counts▼
1Left pass: build left_distinct[i] = # unique chars in s[0..i]
2Right pass: build right_distinct[i] = # unique chars in s[i..n-1]
3Count positions i where left_distinct[i] == right_distinct[i+1]
3The Code (Annotated)▼
def numSplits(s):
n = len(s)
left = [0]*n; seen = set()
for i,c in enumerate(s):
seen.add(c); left[i] = len(seen)
right = [0]*n; seen = set()
for i in range(n-1,-1,-1):
seen.add(s[i]); right[i] = len(seen)
return sum(left[i]==right[i+1] for i in range(n-1))
