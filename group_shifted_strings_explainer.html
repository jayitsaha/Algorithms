Group Shifted Strings

LeetCode #249 – Group strings that belong to the same shift sequence

Medium Hash Map Shift Pattern Key O(n·k) Time
1Understanding the Problem▼

Two strings belong to the same shift group if one can be shifted (all chars +k mod 26) to produce the other. Group all strings from the input.

Example: "abc"→"bcd"→"xyz" all same group. "az"→"ba" same group (wraps around).

💡 Key Insight

Normalize each string by computing differences between consecutive characters mod 26. Two strings in the same shift group will have identical difference sequences → use as hash map key.

2The Code (Annotated)▼
from collections import defaultdict
def groupStrings(strings):
groups = defaultdict(list)
for s in strings:
# key = tuple of consecutive differences mod 26
key = tuple((ord(s[i+1])-ord(s[i]))%26
for i in range(len(s)-1))
groups[key].append(s)
return list(groups.values())
🔐 Why mod 26?

"az" → differences = [(ord('z')-ord('a'))%26] = [25]. "ba" → [(ord('a')-ord('b'))%26] = [(-1)%26] = [25]. Same key → same group ✓
