Two Sum

LeetCode #1 – Find two indices that add up to target using a hash map

EasyHash MapO(n) TimeO(n) Space
1Understanding the Problem▼

Find two numbers in the array that add to target. Return their indices. Exactly one solution guaranteed.

💡 Key Insight

For each number num, its complement is target - num. Store seen numbers in a hash map. If complement is already in map → found! Otherwise store current number and its index.

2The Algorithm▼
1For each i, num: compute complement = target - num
2Check if complement is in the hash map
3If yes → return [map[complement], i]
4If no → add num → i to map
def twoSum(nums, target):
seen = {} # num → index
for i, num in enumerate(nums):
complement = target - num
if complement in seen:
return [seen[complement], i]
seen[num] = i
