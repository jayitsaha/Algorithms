Number of Zero-Filled Subarrays
LeetCode 2348 — Count all contiguous subarrays containing only zeros
🟡 Medium Array Manipulation Math (Consecutive Count) O(n) time O(1) space
1 Understanding the Problem

Given an integer array, count all subarrays that contain only zeros. A subarray is a contiguous non-empty sequence of elements.

Input: nums = [1, 3, 0, 0, 2, 0, 0, 4]
Output: 6
Zero-filled subarrays: [0], [0], [0,0], [0], [0], [0,0] → 6 total
💡 Key Insight

For k consecutive zeros, the number of zero-filled subarrays is:
1 + 2 + 3 + ... + k = k(k+1)/2

But instead of finding each group, we can count as we go: for each zero, add the current consecutive-zero count to our total!

2 The Intuition: Counting Subarrays Ending Here

Instead of grouping zeros and applying a formula, think about it differently: How many new subarrays END at each position?

[...0]
1 subarray ends here
[...0, 0]
2 subarrays end here: [0], [0,0]
[...0, 0, 0]
3 subarrays end here
🧠 The Pattern

For each zero we encounter:

  1. Increment our consecutive-zeros counter
  2. Add that counter to the total (it represents new subarrays ending here)

For non-zeros: reset the counter to 0.

3 The Code
def zeroFilledSubarray(nums):
total = 0 # Total count of zero-filled subarrays
consecutive = 0 # Current consecutive zeros ending here
for num in nums:
if num == 0:
consecutive += 1 # Extend the run
total += consecutive # Add new subarrays ending here
else:
consecutive = 0 # Reset the run
return total
⚠️ Why Add Consecutive, Not 1?

Each zero doesn't just add one new subarray. It adds subarrays of length 1, 2, 3, ..., up to the current consecutive count. Adding the consecutive count each time achieves this automatically!

6 Pattern Recognition & Extensions
🏷️ When to Use This Pattern
  • Counting subarrays with a specific property (all zeros, all same value)
  • The property depends on consecutive elements
  • You need total count, not the actual subarrays
