Understanding the Problem
▼

Find all unique triplets [a,b,c] such that a+b+c=0. No duplicate triplets in result.

💡 Key Insight

Sort first. Fix one element (pivot) and find pairs summing to -pivot using two pointers. Skip duplicates at both the pivot level and after finding triplets.

2
The Algorithm: Sort + Fix One Pivot + Two Pointers
▼

Approach: Sort + Fix One Pivot + Two Pointers

1Sort the array
2For each i from 0..n-3: skip if dup of prev
3 left=i+1, right=n-1
4 Compute sum = nums[i]+nums[l]+nums[r]
5 sum==0: record, skip dups, move both; sum<0: l++; sum>0: r--
3
Why It Works: The Invariant
▼
🔒 Invariant

For fixed pivot nums[i], two-pointer scan finds all pairs in nums[i+1..n-1] summing to -nums[i]. Uniqueness guaranteed by sorting + duplicate skipping.

O(n) pivots × O(n) two-pointer scan = O(n²). Sorting enables two-pointer and natural dedup.

✅ Complexity

Time: O(n²) | Space: O(1)

The Code (Annotated)
▼
def threeSum(nums: list[int]) -> list[list[int]]:
    nums.sort()
    result = []
    for i in range(len(nums) - 2):
        if i > 0 and nums[i] == nums[i-1]:
            continue             # skip dup pivot
        left, right = i+1, len(nums)-1
        while left < right:
            total = nums[i] + nums[left] + nums[right]
            if total == 0:
                result.append([nums[i], nums[left], nums[right]])
                while left < right and nums[left] == nums[left+1]: left += 1
                while left < right and nums[right] == nums[right-1]: right -= 1
                left += 1; right -= 1
            elif total < 0: left += 1
            else: right -= 1
    return result
⚠ Common Mistake

Forgetting to skip duplicates after finding a triplet (inner while loops). Also i>0 check before comparing nums[i]==nums[i-1].
