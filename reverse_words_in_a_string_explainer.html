Reverse Words in a String

LeetCode #151 · Solution 3: Two-Pass Reversal Trick
Medium String Processing Reversal Trick O(n) Time O(1) Extra Space*
1 Understanding the Problem ▼

Given a string containing words separated by spaces (possibly multiple), return the words in reverse order with exactly one space between them. No leading or trailing spaces.

Concrete example:

Input"  the sky  is blue  "
Output"blue is sky the"

We're reversing the order of words, not individual characters. We also need to clean up any extra whitespace.

💡 Key Insight

If you reverse the entire string, the words end up in the right order — but each word is spelled backwards. One more pass to reverse each word individually fixes everything.

2 The Algorithm: Two-Pass Reversal Trick ▼

This classic technique reverses word order using only character-level reversals — no extra arrays or word lists needed.

1Clean spaces — Remove leading, trailing, and duplicate spaces
2Reverse entire string — All characters flip positions. Words are now in correct order but each is reversed
3Reverse each word — Scan for word boundaries, reverse characters within each word

Visual transformation:

Full Reversee u l b · s i · y k s · e h t
Each Word Revb l u e · i s · s k y · t h e
🔑 Why Two Reversals Work

Reversing at two different scopes (whole string vs. individual words) flips only the word order while preserving character order within each word. Think of it like: "Flip the entire sentence, then flip each word right-side up."

3 Why It Works: The Reversal Invariant ▼
🔐 The Invariant

For any sequence of words W₁ W₂ … Wₙ, reversing the full string produces rev(Wₙ) rev(Wₙ₋₁) … rev(W₁). Reversing each word individually then yields Wₙ Wₙ₋₁ … W₁ — the words in reverse order, spelled correctly.

Original: [W₁] [W₂] [W₃] [W₄]
Full reverse: [rev(W₄)] [rev(W₃)] [rev(W₂)] [rev(W₁)]
Word reverse: [W₄] [W₃] [W₂] [W₁] ✓

Edge Cases

  • Single word: Full reverse = word reverse → same result (correct)
  • Extra spaces: Handled by the cleaning step before reversal
  • Already reversed: Algorithm still works — double reversal is identity
⚠️ In Python, Strings Are Immutable

True O(1) extra space only works in languages with mutable strings (C, Java char arrays). In Python, we convert to a list first, adding O(n) space. The algorithm is still O(1) extra — it's the language limitation.

4 The Code (Annotated) ▼

Solution 3 — Two-Pass Reversal Trick:

# Clean spaces → reverse all → reverse each word def reverseWords(s): words = s.split() # Split on any whitespace, removes extras chars = list(' '.join(words)) # Rejoin with single spaces → mutable list def reverse(l, r): # In-place sub-array reversal while l < r: chars[l], chars[r] = chars[r], chars[l] l += 1; r -= 1 # Phase 1: Reverse entire character array reverse(0, len(chars) - 1) # Phase 2: Reverse each individual word start = 0 for i in range(len(chars) + 1): if i == len(chars) or chars[i] == ' ': reverse(start, i - 1) # Reverse word from start to i-1 start = i + 1 # Next word starts after the space return ''.join(chars)

All three solutions at a glance:

# Solution 1 — Split + Reverse + Join (Interview Pick) def reverseWords(s): return " ".join(s.split()[::-1]) # One-liner: O(n) time, O(n) space # Solution 2 — Right-to-Left Two-Pointer def reverseWords(s): result, i = [], len(s) - 1 while i >= 0: while i >= 0 and s[i] == ' ': # Skip spaces i -= 1 if i < 0: break j = i while j >= 0 and s[j] != ' ': # Find word start j -= 1 result.append(s[j+1:i+1]) # Extract word i = j - 1 return " ".join(result)

5 🎤 Interview Strategy: How to Arrive at This ▼
Step 1: Recognize the Pattern

"Reverse the order of groups" is a classic reversal trick signal. Other signals: "in-place", "O(1) space", "rearrange blocks".

Step 2: Start With the Simple Approach

Immediately offer split → reverse → join. This shows you understand the problem and can code quickly. Complexity: O(n) time, O(n) space.

Step 3: Optimize When Asked

"For O(1) extra space, I'd use the reversal trick: reverse the entire string, then reverse each individual word. This works because two reversals at different scopes flip only the word order."

Step 4: Handle Edge Cases

Mention: leading/trailing spaces, multiple spaces between words, single-word input, empty string. Show you handle them in the cleaning step.

🗣️ What to Say to the Interviewer

"I see we need to reverse word order. The quickest approach is split-reverse-join for O(n) time and space. If we need O(1) extra space, we can use the reversal trick: reverse all characters so words are in the right order but backwards, then reverse each word to fix the spelling. I'll use a helper reverse function with two pointers."
