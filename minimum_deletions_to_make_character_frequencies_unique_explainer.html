Minimum Deletions to Make Character Frequencies Unique

LeetCode #1647 – Make all character frequencies distinct with fewest deletions

Medium Greedy Frequency Decrement O(n) Time
1Understanding the Problem▼

Given a string, find the minimum number of character deletions so that all character frequencies are distinct (no two chars have the same frequency).

Example: "aab" → freq: a=2, b=1 → already unique → 0 deletions.

Example: "aaabbbcc" → freq: a=3, b=3, c=2 → reduce one 'b' to 2, but c=2 conflicts → reduce 'b' to 1 → 2 deletions.

💡 Key Insight

For each conflicting frequency, greedily reduce it by 1 until it's either unique or 0. Track used frequencies in a set.

2The Code (Annotated)▼
from collections import Counter
def minDeletions(s):
freq = sorted(Counter(s).values(), reverse=True)
used = set()
deletions = 0
for f in freq:
while f > 0 and f in used:
f -= 1; deletions += 1
if f > 0: used.add(f)
return deletions
